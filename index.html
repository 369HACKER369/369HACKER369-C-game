C++ Knowledge Base

Unit 1: Fundamentals

Variables & Types: int, float, double, char, bool. Store data in memory.

Operators: Arithmetic (+, -, *, /, %), Relational (==, !=, <, >), Logical (&&, ||, !), Bitwise (&, |, ^, ~).

I/O: std::cin >> for input, std::cout << for output.

Unit 2: Control Flow

Conditions: if, else if, else for branching logic. switch for discrete value checks.

Nested: Placing if inside if for complex multi-tier checks.

Unit 3: Loops & Jumps

Loops: for (count-controlled), while (condition-controlled preprocessing), do-while (postprocessing, runs at least once).

Jump Statements: break exits loop/switch. continue skips rest of current loop iteration.

Unit 4: Functions

Functions: Reusable blocks of code. Have return type, name, and parameters. e.g. int add(int a, int b).

Recursion: A function calling itself to solve smaller sub-problems. Needs a base case to avoid infinite loop.

Unit 5: Data Structures

Arrays: Fixed-size sequential memory holding elements of same type. Indexing starts at 0.

Strings: std::string represents text. Can be concatenated with +. Accessed like arrays of chars.
